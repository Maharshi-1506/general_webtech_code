On... Attributes in HTML

The on... attributes in HTML are event attributes that allow you to execute JavaScript code when a specific event occurs on an HTML element. These events can be triggered by user interactions (like clicking, typing, or hovering) or by changes in the browser environment (like page load, window resize, or form submission).

Click events:

  • onclick: Triggered when an element is clicked.
  • ondblclick: Triggered when an element is double-clicked.
  • oncontextmenu: Triggered when the context menu (right-click) is opened.

Mouse events:

  • onmouseover: Triggered when the mouse pointer moves over an element.
  • onmouseout: Triggered when the mouse pointer moves out of an element.
  • onmousemove: Triggered when the mouse pointer is moved within an element.

Keyboard events:

  • onkeydown: Triggered when a key is pressed.
  • onkeypress: Triggered when a key is pressed and released.
  • onkeyup: Triggered when a key is released.

Form events:

  • onsubmit: Triggered when a form is submitted.
  • onreset: Triggered when a form is reset.
  • onchange: Triggered when the value of an input element changes.

Window events:

  • onload: Triggered when the page has finished loading.
  • onunload: Triggered when the page is about to unload.
  • onresize: Triggered when the window is resized.
  • onscroll: Triggered when the window is scrolled.

Other events:

  • onfocus: Triggered when an element gains focus.
  • onblur: Triggered when an element loses focus.
  • onmouseover: Triggered when the mouse pointer moves over an element.
  • onmouseout: Triggered when the mouse pointer moves out of an element.
  • onmousemove: Triggered when the mouse pointer is moved within an element.
